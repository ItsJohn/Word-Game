# Word-Game
This is a Python Project like the game Countdown. The user gets a word that is 7 letters or longer and they have to make 7 words that are 3 letters or greater. For an online example, follow this url: `http://c00176932.pythonanywhere.com/`

# Running locally
1. Download this repo
2. While in the root directory, type `python3 webapp.py`
3. On your browser, go to `http://127.0.0.1:5000/`
